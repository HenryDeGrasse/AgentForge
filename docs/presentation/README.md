# AgentForge Presentation

Terminal slideshow built with [presenterm](https://github.com/mfontanini/presenterm).

## Run

```bash
# Install presenterm (if not already)
brew install presenterm    # macOS
cargo install presenterm   # or from source

# Run from repo root
presenterm docs/presentation/slides.md
```

## Controls

- `→` / `Space` / `Enter` — next slide or reveal
- `←` / `Backspace` — previous slide or reveal
- `q` / `Esc` — quit
